## インストール方法

### 0. discord botの準備をする
ここら辺を参考に  
https://discordjs.guide/preparations/setting-up-a-bot-application.html#creating-your-bot

### 1. .envファイルを作る
```shell
BOT_TOKEN={your bot token}
BOT_CHANNEL={話させたいチャンネル名}
```

### 2. ライブラリインストール
```shell
npm install
```

### 3. 実行
```shell
node index.js
```
